Introduction

This challenge group contains only 10 problems that go over the basics of C-string manipulation. These problems will use basic for-loop control structures as well as conditionals to help you better learn array and string manipulation. Learn more about C-strings through the official course page.

General Suggestions

Be patient and make sure to read each problem description's hints. The first problem set is designed to be overly easy, so you spend most of your time working on getting used to CoreCodex.
